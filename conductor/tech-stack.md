# Tech Stack: SIGECON

## Frontend
- **Framework:** React (TypeScript) - Para uma interface reativa e tipagem forte.
- **Styling:** Vanilla CSS (CSS Modules) - Mantendo a simplicidade e evitando dependências pesadas de UI.
- **Responsividade:** Mobile-first approach, focado em facilitar o uso por funcionários no dia-a-dia.

## Backend
- **Framework:** Python (FastAPI) - Alta performance, documentação automática (Swagger) e facilidade de desenvolvimento.
- **Language:** Python 3.11+
- **Auth:** JWT (JSON Web Tokens) para sessões seguras.

## Database
- **Engine:** PostgreSQL - Confiabilidade para dados relacionais e histórico de auditoria.
- **ORM:** SQLAlchemy ou SQLModel - Para integração fluida com FastAPI.

## Infra & Tools
- **Containerization:** Docker & Docker Compose - Garantindo que o ambiente seja idêntico em qualquer máquina.
- **CI/CD:** GitHub Actions (planejado).
