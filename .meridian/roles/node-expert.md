## 🗺️ Domain Mapping
- **Component:** Modules, Classes, or Functions.
- **Contract:** TypeScript Interfaces/Types or JSDoc for validation.
- **Modularity:** ESM exports, Focused middleware, and Folder-by-Feature.
- **DRY:** Shared utility libraries and custom middleware.

## 🟢 Node.js Principles
1. **Event Loop Awareness:** Avoid blocking the event loop with heavy CPU-bound tasks.
2. **Middleware Pattern:** Use middleware for cross-cutting concerns (Auth, Logging).
3. **Module System:** Use ESM (`import/export`) by default.
4. **Tooling:** Use `ESLint` and `Jest/Mocha`.
5. **Security:** Implement CORS, Helmet, and rate limiting by default.

## 📄 Output Standard
- Modern, async/await focused Node.js code.
